RECURSO

I

O QUE UTILIZAMOS.

Para o projeto Five Devs, foram utilizados
diversos recursos de software e hardware
para garantir a eficiência e qualidade do
produto final.

FIGMA

Plataforma de design de interface do usuário que permite que equipes criem, colaborem, prototipem e compartilhem projetos em um único espaço.Utilizamos ele para prototipar as páginas.

VS CODE

Visual Studio Code é um editor de código-fonte gratuito, de código aberto e multiplataforma que suporta muitas linguagens e recursos. Usamos ele para codar o front-end com HTML e CSS.

ECLIPSE

Eclipse é uma plataforma de desenvolvimento integrado (IDE) de código aberto que suporta várias linguagens de programação. Na parte de FPOO, utilizamos a linguagem Java.

UBUNTU LINUX

Ubuntu é uma plataforma de software livre que pode ser executada em qualquer lugar, desde o PC até o servidor e a nuvem.

WINDOWS 10

Ubuntu é uma plataforma de software livre que pode ser executada em qualquer lugar, desde o PC até o servidor e a nuvem.
