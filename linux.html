COMO BAIXAR O LINUX.

Neste módulo, ensinaremos passo a passo
como instalar o Unbuntu, que é um Sistema
Operacional de código aberto baseado em Lixus.

requisitos

Antes de partir para a instalação do Ubuntu, você precisa saber se tem:

  1. 25 GB de espaço de armazenamento livre para instalação do sistema;
  2. 4 GB de memória RAM;
  3. Processador dual-core de, no mínimo, 2 GHz;
  4. Uma porta USB;
  5. Um pendrive de, no mínimo, 8 GB (se tiver 12 ou mais é melhor).
